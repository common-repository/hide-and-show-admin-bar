Adds a toggle button to admin bar in the front-end to hide and show the admin bar when required.

== Description ==
Hide and Show Admin Bar plugin add a toggle button in the front-end with which you can hide the admin bar when you don't need it and you can view admin bar when want it. This gives the flexibility to view clean site and have access to admin bar.

Toggle button remains visible to access the admin bar.

If you have any comments or questions, please use the support forum: http://wordpress.org/support/plugin/hide-and-show-admin-bar

== Installation ==

1. Upload `hide-and-show-admin-bar` directory to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Once activated you will see the toggle button top right corner of the screen.
